# flask-todo-app
# Flask To-Do List Web Application

## 📌 Project Description

This project is a simple To-Do List web application developed using **Python Flask** and **SQLite database**.
It allows users to add, view, and delete daily tasks through a web interface.

---

## 🎯 Objectives

* To understand the basics of Flask framework.
* To implement CRUD operations using SQLite.
* To design a simple frontend using HTML & CSS.
* To connect backend with database.
* To deploy project using GitHub.

---

## 🛠 Technologies Used

* Python 3
* Flask
* SQLite3
* HTML5
* CSS3
* Git & GitHub

## ⚙️ How to Run the Project

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000/
